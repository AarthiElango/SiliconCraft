FPGA-Based System Design

Design, integrate, and validate FPGA subsystems using Verilog/SystemVerilog and vendor IP. Implement DMA/AXI pipelines and real-time processing on FPGA boards.

Skills Covered

  • RTL coding, constraints & timing
  • IP integration, AXI/DMA, memory-mapped I/O
  • On-board debug (ILA/Chipscope)
  • Image/signal processing & acceleration

Projects

  • ISP pipeline on FPGA
  • AXI DMA frame buffer & display
  • DDR4, PCIe, ADC/DAC, Camera Interface
  • Realtime image signal processing

Job Roles

  • FPGA Design/Validation Engineer
  • Hardware Accelerator Engineer
  • Prototyping/Bring-up Engineer

Career Pathway

Trainee → Jr. FPGA → FPGA Engineer → Sr. FPGA → Lead
